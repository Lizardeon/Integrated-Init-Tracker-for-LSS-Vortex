# Integrated Init Tracker for LSS Vortex

**Integrated Init Tracker** — это пользовательский скрипт (UserScript) для расширения [Tampermonkey](https://www.tampermonkey.net/), предназначенный для ведения боевых столкновений в веб-приложении [Long Story Short (Vortex)](https://vortex.longstoryshort.app/).

Инструмент добавляет полноценный трекер инициативы прямо в интерфейс приложения, позволяя ДМу быстро управлять ходами, импортировать игроков со «Стола» и отслеживать состояние NPC.

## Основные возможности

* **Интеграция**: Скрипт добавляет новую вкладку «Трекер Инициативы» в панель инструментов Vortex.
* **Импорт игроков**: Автоматическое считывание персонажей со «Стола». Скрипт пытается найти последнюю брошенную игроком инициативу в логе чата, чтобы автоматически заполнить порядок хода.
* **Управление NPC**: Удобная форма для добавления монстров и NPC с настройкой HP, КД и инициативы.
* **Гибкое управление HP**:
* Быстрые команды для урона (`-10`) и лечения (`+10`).
* Поддержка временных хитов (THP).
* Автоматическая цветовая индикация состояния здоровья.


* **Состояния и эффекты**: Возможность добавлять и удалять статусы/дебаффы для каждого NPC.
* **Персистентность**: Все данные боя сохраняются в `localStorage` браузера, поэтому состояние трекера не сбросится при обновлении страницы.

## Установка

1. Установите расширение **Tampermonkey** для вашего браузера (Chrome, Firefox, Edge и др.).
2. Нажмите на иконку Tampermonkey и выберите «Создать новый скрипт» (Create a new script).
3. Скопируйте весь код из файла `Integrated Init Tracker for LSS Vortex.user.js` в редактор (или просто перетяните в окошко со всеми скриптами)
4. Нажмите `Ctrl + S` (или «Файл» -> «Сохранить»).
5. Откройте [Vortex](https://vortex.longstoryshort.app/) — вкладка трекера появится автоматически.

## Использование

1. **Начало боя**: Нажмите кнопку **«Новый бой»** в трекере. Скрипт просканирует карточки игроков на «Столе» и предложит выбрать, кто участвует в столкновении.
2. **Управление ходом**: Используйте кнопки «Назад» и «След. ход» для перемещения по списку инициативы. Раунд увеличивается автоматически после прохождения списка.
3. **Редактирование**: Вы можете кликать по любым числовым полям в таблице, чтобы изменить хиты, КД или инициативу.
4. **Статусы NPC**: Нажмите на кнопку `+` в колонке состояний, чтобы добавить новый эффект. Клик по существующему статусу удалит его.

## Технические детали

* **Стек**: Vanilla JS, CSS.
* **Хранение**: Используется `localStorage` браузера с ключом `vortex_dm_state_v10`.
* **Совместимость**: Скрипт использует `MutationObserver` для отслеживания изменений DOM в интерфейсе Vortex, чтобы корректно встраиваться при переключении вкладок.
